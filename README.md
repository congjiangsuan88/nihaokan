# 🌸 你好看 (Nihaokan) 

> **每日审美 · 治愈系名句壁纸生成器**

「你好看」是一个极简、纯粹的在线海报生成工具。它旨在通过精选的治愈系文案与高质量的审美背景，为用户在忙碌的生活中提供一瞬间的慰藉。

✨ **[立即在线体验](https://congjiangsuan88.github.io/nihaokan/)**

---

### 🌟 核心特性

- **一键治愈**：动态调用金句 API，匹配高质量随机背景图。
- **纯前端驱动**：基于 `html2canvas` 实现毫秒级本地海报合成，无需后端服务器。
- **移动端优化**：完美适配微信、手机浏览器，支持长按保存图片的“最后一公里”交互。
- **极简审美**：无多余 UI 干扰，呼吸感进度条，聚焦内容本身。
- **打赏闭环**：内置优雅的赞赏引导提示，支持变现裂变能力。

### 🛠️ 技术栈

- **Core**: HTML5 / CSS3 / JavaScript (Vanilla JS)
- **Library**: [html2canvas](https://html2canvas.hertzen.com/)
- **API**: [Hitokoto (一言)](https://hitokoto.cn/) / [Unsplash (Lorem Picsum)](https://picsum.photos/)

### 🚀 快速上手

1. **Fork** 本仓库。
2. 将 `index.html` 中的赞赏码 Base64 替换为你自己的。
3. 开启 **GitHub Pages** 即可上线。
